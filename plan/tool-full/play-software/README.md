## How to make deb-src package


### help

``` sh
$ make
```

or

``` sh
$ make help
```


### prepare

``` sh
$ make prepare
```


### build deb

``` sh
$ make deb-build
```


### install deb

``` sh
$ make deb-install
```


### remove deb

``` sh
$ make deb-remove
```


### clean deb

``` sh
$ make deb-clean
```

## Debug Mode

``` sh
$ export DEBUG_PLAY_SOFTWARE=true
```
