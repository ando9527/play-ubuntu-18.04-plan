
# Quick Start


## install git

``` sh
$ sudo apt-get install git
```


## git clone

``` sh
$ git clone https://github.com/samwhelp/play-ubuntu-18.04-plan.git
```

## cd root dir

``` sh
$ cd play-ubuntu-18.04-plan
```


## cd play-lxqt dir

``` sh
$ cd plan/de-basic/play-lxqt
```


## make prepare

``` sh
$ make prepare
```


## make deb-build

``` sh
$ make deb-build
```


## make deb-install

``` sh
$ make deb-install
```


## mod-lxqt help

``` sh
$ mod-lxqt
```

or

``` sh
$ mod-lxqt help
```


## mod-lxqt install


``` sh
$ mod-lxqt install
```
